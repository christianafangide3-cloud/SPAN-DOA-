# Security Policy for SPAN-DOA

## Supported Versions
Only the current V2 release of the SPAN-DOA Portal is supported with security updates. Legacy V1 is deprecated.

## Reporting a Vulnerability
As a medical-grade platform, we take security seriously. If you discover a vulnerability or a bypass in the CBT engine, please **do not create a public issue**. 

Instead, report it directly to the Director of Academics:
**Email:** splendidafangide@gmail.com

You should expect a response within 24 hours. Verified vulnerability reports will be credited in the Hall of Fame.
